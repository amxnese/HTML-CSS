Here are Some Languages and its Uses:

HTML
Stands for Hyper Text Markup Language.
The standard markup language for creating Web pages.
describes the structure of a Web page.
consists of a series of elements that tell the browser how to display the content.
CSS
stands for Cascading Style Sheets.
describes how HTML elements are to be displayed on screen, paper, or in other media.
saves a lot of work. It can control the layout of multiple web pages all at once.
External stylesheets are stored in CSS.
Javascript
JavaScript is a lightweight, interpreted programming language.
Designed for creating network-centric applications.
